# Quick Start

Start with the main guide:

- [Quick start](../../what-is-this/quick-start.md)

If you are starting from scratch, the fastest sequence is:

1. Browse modules.
2. Create an account.
3. Add your modules to your collection.
4. Build a rack.
5. Capture a patch.

Then continue with:

- [Modules](../../learn-patcher.xyz/modules.md)
- [Collection](../../learn-patcher.xyz/collection.md)
- [Racks](../../learn-patcher.xyz/racks.md)
- [Patches](../../learn-patcher.xyz/patches.md)
