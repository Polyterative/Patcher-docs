# Quick start

## Create a patch

* create an account
* navigate to “modules”
* open your module
* click on the “+” button to add to your collection
* navigate to “collection”
* click “Create Patch”
* Open your new patch

## Add modules to collection

* create an account
* navigate to “modules”
* open your module
* click on the “+” button to add to your collection

## Create a patch

* add modules to your collection
* navigate to “collection”
* click “Create Patch”
* Open your new patch

## Add connections to a patch

* Create a patch
* Add modules to collection
* Verify that modules already have tracked INs and OUTs (add if missing)
* Click on the INs and OUTs to select a connection
* Confirm connection
* Save patch
